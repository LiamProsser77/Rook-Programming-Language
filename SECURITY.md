Security Policy
Reporting Security Vulnerabilities
HPP takes security seriously. We appreciate the efforts of security researchers and users who help improve the safety of the project.

If you discover a potential security vulnerability, please report it responsibly.

How to Report
Please do not disclose security issues publicly.

You can report security vulnerabilities by:

Opening a private discussion on GitHub, or
Contacting the project maintainer directly through GitHub
We kindly ask that you provide sufficient detail to help us understand and reproduce the issue.

Responsible Disclosure
We aim to review reported security issues in a timely manner and will take appropriate steps to address them. Responsible disclosure helps ensure the safety and integrity of HPP and its users.

Thank you for helping keep the project secure.
